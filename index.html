Shima, ADB ke through App Install Karne ka Asaan Tarika (Rooted Phone ke liye)

Tumhare phone mein root hai, isliye bina PC ke hi ADB commands se app install kar sakti ho. Neeche diye gaye steps dhyan se follow karo:

Step 1: Shizuku Install Karo

  • Play Store se Shizuku install karo.
  • Install from Play Store

Step 2: AShell Install Karo

  • Play Store se AShell app install karo.
  • Install from Play Store

Step 3: Shizuku Start Karo

  • Shizuku app open karo.
  • "Start" button dabao. (Rooted phone hone ki wajah se direct start ho jaayega)

Step 4: AShell Open Karo aur Permission Do

  • AShell app open karo.
  • Jab Shizuku permission maange, allow kar dena.

Step 5: APK File Copy Karo (Command #1)

cp "/storage/emulated/0/Download/CapCut v14.3.0 (Pro) (Login Required).apk" /data/local/tmp/

Step 6: APK Install Karo (Command #2)

pm install --user 12 "/data/local/tmp/CapCut v14.3.0 (Pro) (Login Required).apk"

Note: Agar --user 12 se kaam na chale to --user 0 try karna

Step 7: Wait Karo

Command complete hone ke baad app install ho jaayega.
